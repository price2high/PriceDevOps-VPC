# Frontend Setup: Static Website on S3

The frontend is a lightweight HTML/CSS site designed to showcase Price DevOps services and collect client booking requests.

## Steps

1. Create a new S3 bucket
2. Enable static website hosting
3. Upload `index.html` and `booking.html`
4. Set bucket policy for public read access
5. Copy static endpoint URL (or connect via Route 53)

## Files

- `index.html`: Overview of Price DevOps services
- `booking.html`: Appointment form with POST action to backend API
